# LimeUI
Free simple UI system for Game Maker games

Features:
- Custom styles
- Containers
- Panels
- Scroll panels
- Buttons
- Checkboxes
- Texts
- Textboxes
- Progress bars
- Sliders
- Spriteboxes
- Sprite buttons
- Dropdown lists

In development now:
- Tabs panel
- Draggable windows
- Animations
